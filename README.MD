App Description
This is google chrome extension for a Twitch chat 7tv. It reads the url to get the streamers username, sends it to decapi.me to resolve the user-id needed to query the streamer's 7tv emotes. The extension creates a button at the bottom right of the Twitch chat and clicking it will toggle open and close a menu containing all the emotes, a search textbox box, and a textbox to load any other streamer's emotes in the menu. When any emote is clicked, then the emote name will be put into the chat window. The menu is resizable on height by moving a bar on the top of the menu.
